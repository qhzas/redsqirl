File Text Source

Overview

File Text Source is an action that imports a plain text source character delimited file.

Example:
File contains the following: "ab,c", abc
Delimiter set to ","
File text source would create a record with the following three fields: "ab , c" and abc

  • Input
    • PPlain text source character delimited file
  • Output
    • Plain Text Map Reduce File

Source: Data Sub-Type

Format

The user can select the format to use. The user must choose if the text file has a header or not.

Source: Browser

Data set

Within the dataset browser the user can select the path of the data that will be used.

Data format parameters

The user can change the delimiter. The delimiter can be a normal character (",","|",";") or an ASCII character in which case (#1, #44).

Data preview

The user can change the header and fields of a file by clicking on the pencil Icon. A header can be a list of feature names or a list of feature name with type. For example, if the user has a data set that contains three fields, The user could define the header to be "field1, field2, field3" or "field1 DATE, field2 STRING, field3 INT".

Buffer a dataset

Our workflow is almost ready to run. However, the Spark Aggregator produces a TEMPORARY output (bottom right arc is purple). The output type will be deleted as soon as the next action is run. Therefore, once the notification is sent, the data will be deleted! So we need to tell Red Sqirl to keep it.

  1. Select the Spark Aggregator action.
  2. In the top menu go to Edit > Output state > Buffered. The bottom right arc of the action should change to blue.
    • A click on the canvas is sometimes necessary for this change to take place.

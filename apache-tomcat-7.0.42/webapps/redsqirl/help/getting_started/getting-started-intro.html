Getting Started - Introduction

Welcome to the Red Sqirl getting started tutorial! This tutorial takes approximately 20 minutes to complete. Once you have explored the aspect highlighted in one task, please click next to continue. If the help section is too small or too big, you can resize the help at any time by dragging the vertical slider.

Red Sqirl is an analytics platform for Big Data. The main workspace let the user build analytic pipelines. Red Sqirl is actually a framework and leverages a marketplace from which user pull powerful tools. This getting started guide uses Apache Spark, please check if you have a "Spark-ETL" menu in your footer on the bottom right of your screen. Alternatively Pig can be used instead.

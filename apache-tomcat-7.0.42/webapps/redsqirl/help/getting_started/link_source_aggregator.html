Link the source with the aggregator

Before configuring an action that requires an input dataset, it is always necessary to link it first. Here we need to declare our Source nodes to be the input data of our aggregator. To create a link:

  1. You can click in the space between the icon and the arc. You should notice the mouse cursor will change in the area where you can click.
  2. An arrow will follow your cursor, click a second time on the Spark Aggregator icon to create the link.

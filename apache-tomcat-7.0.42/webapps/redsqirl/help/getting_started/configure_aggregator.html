Configure the Spark aggregator

The first page of the aggregator lets you choose the common attribute on which the data is aggregated. The second page lists the output fields you would like to produce.

  1. Select "PROP_VALUE" and move it to the right column, then click next.
  2. The generator lists the most common calculations. Click on OK, next to copy. This action will generate a line in your table "PROP_VALUE".
  3. Select count in the drop down generation menu and click OK again. That way we will count the number of non null "PROP_NAME" values as well as the number records for every "PROP_VALUE".
  4. Click OK on the bottom right to close the window.

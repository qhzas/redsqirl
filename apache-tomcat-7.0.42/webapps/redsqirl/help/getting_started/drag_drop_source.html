Drag & Drop a source

When running a workflow, Red Sqirl submits a job to an open source project called Apache Oozie. The code generated is available on the Hadoop file system and therefore can be reused. Here, we will take a file containing variables from our previous job, and summarize its content.

Red Sqirl supports many type of sources. Every type have its supported formats. In this example, our data is on the Hadoop file system and is a plain text file, therefore we will use a "File Text Source".

  1. Drag and Drop a File Text source
  2. Double click on it
  3. Name the action "rs_prop_file"
  4. Click OK

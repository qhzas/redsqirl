Choose a path

Red Sqirl generates workflows in the user Hadoop file system in .redsqirl/jobs/$canvas_name_$run_cnt

  1. Select HDFS Text File (our file does not have any header).
  2. Click Next.
  3. Click on the magnifying glass icon
  4. Type in the path text field /user/{user_name}/.redsqirl/jobs/getting_started_0
  5. Select the "job.properties" file
  6. Click OK

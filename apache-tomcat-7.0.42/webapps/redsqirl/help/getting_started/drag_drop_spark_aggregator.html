Drag & Drop a Spark aggregator

Let's summarize our very tiny dataset, just so we can understand the tool at hand.

Spark is an 'in memory technology' that is famous for its machine learning library, but it can also do ETL. If Spark is not available, use the Pig Aggregator action in the Pig footer menu (pink icon).

  1. Click on Spark-ETL on the footer menu.
  2. Drag and drop a Spark Aggregator node.

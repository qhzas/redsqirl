Send Email

Overview

This action sends an email. An email can be sent at the beginning of a workflow or after several actions are completed. When the action is used at the end of a workflow, the user have to buffer the user's result or record them as they will be removed otherwise.

  • Input
    • Text or Binary Map Reduce Directory
  • Configurations
    • Receiver
    • Carbon Copy Receiver
    • Subject
    • Email body

Email

Fill out the form as if the user were sending an email from the user's mailbox.

HCatalog Source

Overview

HCatalog Source is an action that defines a data source for a workflow. A data source used in this node can come from a type of storage facility. There are two types of storage facilities are Hive and HDFS.

  • Inputs
    • No Inputs
  • Outputs
    • Data selected through browser

Source: Browser

Data set browser

Within the dataset browser the user can select the path of the data that will be used.

Data preview

The user can change the field type from CATEGORY to STRING by clicking on the pencil Icon.

Convert File Text

Overview

Convert File Text aggregates a MapReduce Directory to a plain text file. This action should be used only on small datasets.

  • Input
    • Text Map Reduce Directory
  • Output
    • Text File

Data Output

Output type

The user can define where the text file being created is temporary, buffered or recorded.

Path

Within the Path field the user can select the path where the data that will be stored.

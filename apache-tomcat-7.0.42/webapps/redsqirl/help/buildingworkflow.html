Tutorials

There are three main windows that make up the basic Red Sqirl layout which can be seen in the image below.
Workflows are built by dragging and dropping icons onto the canvas area and joining them together to perform an action.


To get started with Red Sqirl, we recommend the user watch the first step tutorial video. While this video has been recorded using Red Sqirl v0.10 the basic functionality remains the same and will guide the user through the layout and basics of Red Sqirl. It is also highly recommended that the user complete the tutorial offered at startup to also achieve this understanding of the basics.

Synchronous Sink Filter

Overview

Synchronous Sink Filter is an action that reuses a Sink output into another coordinator job. This node allows a data pipeline in one workflow.

  • Inputs - The Synchronous Synk output.
  • Template path - The path in which the data is stored
  • Outputs - The selected instances

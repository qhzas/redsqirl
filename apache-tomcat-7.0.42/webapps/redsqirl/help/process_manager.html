Process Manager

The Process Manager is a way to view the list of jobs that have been submitted to Oozie . It is possible to kill processes that are running. The Oozie Web Console can be accessed in a new window by clicking which will elaborate more details about the jobs being submitted to Oozie. The resource manager offers a low level view of the jobs and a gauge of how busy the cluster is. It can be seen in a new window by clicking .

A job can also be manipulated from this tab

  • Kill: Stop a job. The current running action status will be set to 'kill'.
  • Suspend: Suspend the job until you resume it. It is useful if the cluster is busy and you want to prioritize the jobs.
  • Resume: Resume a suspended job.
